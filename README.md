# wheelight
Light around the tires<br />
Started in August 2015.<br />

1. How to build LED blink example:
    * In the top directory, `mkdir build`
    * `cd build`
    * `cmake ../cmake`
    * `make -j4 blink` <-- this builds the example
    * Connect the PIC 44-pin board.
    * `make pic_example`
